# MyNotes App 📝

**MyNotes** adalah aplikasi pencatatan (*notes*) berbasis *fullstack* yang dirancang dengan antarmuka modern dan *user-friendly* untuk mempermudah penggunadalam mencatat aktivitas harian, ide kreatif, maupun materi pembelajaran. 

Proyek ini dibangun menggunakan **HTML, CSS, dan Vanilla JavaScript** untuk sisi *frontend*, serta **Node.js dan Express.js** untuk sisi *backend*. Aplikasi ini menggunakan sistem basis data relasional **MySQL**. 

Sebagai bagian dari pengembangan infrastruktur, proyek ini juga mencakup praktik migrasi *database* dari lingkungan lokal (*localhost*) menuju komputasi awan (*cloud*) menggunakan layanan **Cloud SQL** atau **Virtual Machine (VM)**.

### 🚀 Tech Stack
- **Frontend:** HTML5, CSS3, Vanilla JavaScript
- **Backend:** Node.js, Express.js
- **Database:** MySQL (Lokal & Cloud SQL/VM)

### ✨ Fitur Utama
- Tampilan UI/UX responsif.
- *Create, Read, Update, Delete* (CRUD) catatan secara *real-time*.
- Pop-up profil interaktif.
